# protein-characterization-tp53
Comprehensive bioinformatics analysis of the human TP53 protein using UniProt, AlphaFold, PDB, IntAct and Reactome.

🧬 TP53 Protein Characterization Using Bioinformatics Databases

A comprehensive bioinformatics case study of the human **TP53 (Tumor Protein p53)** using publicly available biological databases.


## 📖 Overview

TP53 is one of the most extensively studied tumor suppressor proteins and plays a critical role in maintaining genomic stability. This project explores the structure, function, regulation, and biological significance of TP53 through protein annotation databases and structural resources.



## 🎯 Objectives

- Characterize the TP53 protein
- Explore protein domains and functional regions
- Study post-translational modifications (PTMs)
- Analyze protein structure using AlphaFold and PDB
- Investigate protein–protein interactions
- Understand disease associations and biological pathways



## 🛠️ Databases Used

| Database | Purpose |
|----------|----------|
| UniProt | Protein annotation |
| AlphaFold | Predicted 3D protein structure |
| Protein Data Bank (PDB) | Experimental protein structures |
| IntAct | Protein interaction network |
| Reactome | Biological pathways |
